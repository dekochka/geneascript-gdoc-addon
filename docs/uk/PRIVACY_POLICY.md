# Політика конфіденційності — GeneaScript «Транскриптор метричних книг»

**Останнє оновлення:** 2026-04-11

GeneaScript Metric Book Transcriber («Додаток») — це додаток з відкритим кодом для Google Docs™, який допомагає користувачам транскрибувати зображення метричних книг (акти народження, шлюбу та смерті) за допомогою API Google™ AI (Gemini™).

## Які дані обробляє додаток

| Дані | Навіщо | Куди потрапляють |
|------|--------|------------------|
| **Ключ API Google™ AI (Gemini™)** | Потрібен для виклику Gemini™ від імені користувача. | Зберігається в **User Properties** Google Apps Script (приватно для кожного облікового запису Google). Ніде більше не передається, окрім кінцевої точки Generative Language API Google. |
| **Мова інтерфейсу (необов’язково)** | Дозволяє зафіксувати мову UI додатка (англійська, українська, російська) замість мови облікового запису Google. | Зберігається в **User Properties** (`UI_LOCALE`). Лише для вибору рядків інтерфейсу; третім особам не передається. |
| **Вміст документа (розділ Context)** | Текст під заголовком «Context» у активному документі Google Doc читається для побудови запиту на транскрипцію. | Надсилається до API Google™ AI (Gemini™) як частина запиту. Деінде не зберігається. |
| **Вибране зображення** | Зображення, яке користувач обирає для транскрипції, зчитується з документа. | Надсилається (у base64) до API Google™ AI (Gemini™). Деінде не зберігається. |
| **Обрані у Google Drive™ файли (імена та зображення)** | Під час «Import Book from Drive Files» додаток читає лише ті файли зображень, які користувач явно обирає через Google Picker. Використовується OAuth-область `drive.file`: доступ **лише до обраних файлів**, не до всього Drive. | Зображення вставляються в документ Google Doc. Метадані та двійкові дані обробляються лише в пам’яті; файли не кешуються та не зберігаються окремо. |

## Зберігання даних

- **Ключ API** зберігається в User Properties Google Apps Script і приватний для кожного облікового запису Google.
- Додаток **не веде** зовнішню базу даних, сервер чи постійне сховище поза User Properties Apps Script.
- Додаток **не зберігає** вміст документів, зображення чи результати транскрипції поза власним Google Doc користувача.

## Захист даних

- **Шифрування під час передачі:** дані до API Google передаються через HTTPS/TLS.
- **Шифрування та контроль доступу** у системах Google (User Properties, Docs/Drive) регулюються політиками безпеки платформи Google.
- **Мінімізація даних:** обробляються лише дані, потрібні для дій користувача (вибране зображення, текст контексту, обрані для імпорту файли).
- **Без зовнішнього збереження контенту** у базах або сховищах третіх сторін під контролем розробника.
- **Видалення:** користувач може змінити або видалити збережений ключ API; вміст документа залишається під контролем користувача в Google Docs/Drive.

## Сторонні сервіси

Додаток надсилає дані до одного стороннього сервісу в цьому сенсі:

- **Google™ AI (Gemini™) API** (`generativelanguage.googleapis.com`) — для транскрипції зображення. У запиті — текст розділу Context і вибране зображення. Умови обробки даних регулює Google; див. [Політику конфіденційності Google AI](https://ai.google.dev/terms).

Додаток **не** надсилає дані до інших сторонніх сервісів.

## Використання даних Gemini

- Надсилається лише мінімум даних, потрібний для транскрипції; додаток **не** використовує ці дані для навчання власних моделей.
- Подальша обробка даних через Gemini™ залежить від вашого тарифу та умов облікового запису/проєкту Google.
- Ознайомтеся з чинними умовами: [Google AI for Developers](https://ai.google.dev/terms), [Google Cloud terms](https://cloud.google.com/terms).

## Операційна телеметрія

Додаток записує легку операційну телеметрію в **Google Cloud Logging** (проєкт Google Cloud розробника) для діагностики помилок і моніторингу якості. У логах можуть бути:

- тип події (початок транскрипції, завершення імпорту, помилка);
- час і затримки;
- оцінка токенів і вартості для викликів Gemini API;
- коди помилок і **санітизовані** повідомлення (до 300 символів, без вмісту користувача);
- **анонімізований ідентифікатор** — SHA-256 від тимчасового ключа сесії; ідентифікувати конкретного користувача за ним неможливо.

У логах **немає** тексту документа, зображень, імен файлів, ключів API чи ПІБ. Термін зберігання — згідно з налаштуваннями Google Cloud Logging.

Додаток **не** використовує файли cookie, сторонню веб-аналітику чи трекінг.

## Передача даних третім особам

Додаток **не** продає, не передає й не переносить дані користувачів третім особам, окрім описаних викликів API Google AI за ініціативою користувача.

## Контроль користувачем

- Видалити збережений ключ API можна через діалог додатка або технічно в консолі редактора Apps Script.
- Додаток можна видалити в будь-який час — це прибирає пов’язані User Properties.
- Обробка даних відбувається лише після явної дії користувача (транскрипція, пакетна транскрипція з бічної панелі, імпорт з Drive). Фонової роботи немає.

## Діти

Додаток не призначений для дітей молодше 13 років і свідомо не збирає персональні дані дітей.
