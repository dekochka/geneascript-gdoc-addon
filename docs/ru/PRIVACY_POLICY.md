# Политика конфиденциальности — GeneaScript «Транскрибатор метрических книг»

**Последнее обновление:** 2026-04-11

GeneaScript Transcriber («Дополнение») — это дополнение с открытым исходным кодом для Google Docs™, помогающее расшифровывать изображения метрических книг (акты рождения, брака и смерти) с помощью API Google™ AI (Gemini™).

## Какие данные обрабатываются

| Данные | Зачем | Куда попадают |
|--------|-------|---------------|
| **Ключ API Google™ AI (Gemini™)** | Нужен для вызова Gemini™ от имени пользователя. | Хранится в **User Properties** Google Apps Script (приватно для аккаунта Google). Никуда больше не отправляется, кроме конечной точки Generative Language API Google. |
| **Язык интерфейса (необязательно)** | Позволяет зафиксировать язык UI (английский, украинский, русский) вместо языка аккаунта Google. | Хранится в **User Properties** (`UI_LOCALE`). Только для строк интерфейса; третьим лицам не передаётся. |
| **Содержимое документа (раздел Context)** | Текст под заголовком «Context» в активном Google Doc читается для запроса на расшифровку. | Отправляется в API Google™ AI (Gemini™) как часть запроса. Иначе не хранится. |
| **Выбранное изображение** | Изображение, выбранное для расшифровки, считывается из документа. | Отправляется (в base64) в API Google™ AI (Gemini™). Иначе не хранится. |
| **Выбранные в Google Drive™ файлы** | При «Import Book from Drive Files» дополнение читает только изображения, явно выбранные через Google Picker. Используется область OAuth `drive.file`: доступ **только к выбранным файлам**, не ко всему Drive. | Изображения вставляются в Google Doc. Метаданные и двоичные данные обрабатываются в памяти; отдельно не кэшируются и не сохраняются. |

## Хранение

- **Ключ API** хранится в User Properties и приватен для каждого аккаунта Google.
- Дополнение **не ведёт** внешнюю БД, сервер или постоянное хранилище вне User Properties Apps Script.
- **Не хранит** содержимое документов, изображения или результаты расшифровки вне собственного Google Doc пользователя.

## Защита данных

- **Шифрование при передаче:** HTTPS/TLS.
- **Платформенная безопасность** Google для User Properties, Docs/Drive.
- **Минимизация:** обрабатываются только данные, нужные для действий пользователя.
- **Без внешнего сохранения контента** в БД или хранилищах третьих лиц под контролем разработчика.
- **Удаление:** пользователь может изменить или удалить ключ API; содержимое документа остаётся под контролем пользователя в Google.

## Сторонние сервисы

Данные отправляются в:

- **Google™ AI (Gemini™) API** (`generativelanguage.googleapis.com`) — для расшифровки. Условия обработки — у Google: [Политика конфиденциальности Google AI](https://ai.google.dev/terms).

Другие сторонние сервисы **не** используются.

## Использование данных Gemini

- Передаётся минимум данных; дополнение **не** использует их для обучения собственных моделей.
- Дальнейшая обработка зависит от тарифа и условий вашего проекта/аккаунта Google.
- См. [Google AI for Developers](https://ai.google.dev/terms), [Google Cloud terms](https://cloud.google.com/terms).

## Операционная телеметрия

Записи в **Google Cloud Logging** могут включать: тип события, время и задержки, оценку токенов и стоимости, коды ошибок и **санитизированные** сообщения (до 300 символов, без пользовательского контента), **анонимизированный идентификатор** (SHA-256 от временного ключа сессии).

В логах **нет** текста документа, изображений, имён файлов, ключей API или ПДн в явном виде.

**Нет** cookie, сторонней аналитики и трекинга.

## Передача третьим лицам

Дополнение **не** продаёт и не передаёт данные пользователей третьим лицам, кроме описанных вызовов API Google AI по инициативе пользователя.

## Контроль пользователя

- Удаление ключа API — через диалог дополнения или консоль редактора Apps Script.
- Удаление дополнения — удаляет связанные User Properties.
- Обработка только после явного действия пользователя; фоновой работы нет.

## Дети

Дополнение не предназначено для детей младше 13 лет и сознательно не собирает их персональные данные.

## Изменения политики

Обновлённая версия публикуется в репозитории с новой датой.
